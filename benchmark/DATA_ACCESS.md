# Gastro4D-USSim dataset access

The full simulation archive (monitor/scanner streams, 4D GT surfaces, frozen manifests, and reproduction assets) is distributed separately from this code repository.

## How to request access

1. Download the data access agreement:  
   [`docs/Gastro4D-USSim_Data_Access_Agreement.pdf`](../docs/Gastro4D-USSim_Data_Access_Agreement.pdf)

2. Print the agreement, sign it, and scan it as a **single PDF** file.

3. Email the signed document to **nancy_l_l@163.com** (Liu Yanan).

   Please include in the email:
   - your name and affiliation;
   - a brief description of the intended research use;
   - your Baidu Netdisk account (recommended) or preferred contact email.

4. If approved, you will receive **Baidu Pan download instructions** by email.

## What is included in the archive

- Dual-stream simulated acquisitions (monitor + scanner)
- 4D GT gastric surfaces and phase metadata
- Frozen `dev` / `test` manifests
- Observation-protocol variants (Clean / Sparse / PoseNoise / ImageNoise)

Benchmark scale: 281 instances, 1,124 cases. See `dataset_summary.json`.

## Citation

Please cite the accompanying MedIA manuscript when using Gastro4D-USSim.
